## üìà Quick Access

### View Results

**HTML Viewers (Recommended):**
```bash
# Open in browser
open viewers/step1_viewer.html
open viewers/step1_5_viewer.html
open viewers/step2_viewer.html
```

**Summary Report:**
```bash
open summary/summary_report.html
```

### Find Specific Outputs

**All figures:**
```bash
find results -name "*.png" -type f
```

**All tables:**
```bash
find results -name "*.csv" -type f
```

**Validation reports:**
```bash
cat results/validation/final_validation_report.txt
```
